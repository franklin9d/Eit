طريقة التشغيل:
1) فك الضغط.
2) ارفع المجلد كما هو إلى Vercel أو أي استضافة static.
3) الصفحة الرئيسية: index.html

ملاحظات:
- البيانات محفوظة محلياً داخل data/countries.json
- عدد الدول/المناطق: 249
- عدد الوصفات: 2490
- لكل دولة/منطقة 10 وصفات
- بعض الدول تحتوي وصفات محلية مباشرة (مثل العراق ومصر وتركيا والهند واليابان وإيطاليا وغيرها)
- بقية الدول مغطاة عبر وصفات ممثلة للإقليم الغذائي الأقرب لها حتى لا تبقى أي دولة بلا محتوى
- روابط الفيديو والبحث الخارجي موجودة داخل كل وصفة
